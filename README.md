
# 🚗 Uber Landing Page Clone

A responsive **Uber Landing Page UI Clone** built using **HTML5** and **CSS3**.
This project recreates the modern Uber website interface focusing on layout design, responsiveness, and clean UI implementation.

---

## 📌 Project Overview

This project is a front-end implementation of the Uber ride booking landing page. It demonstrates:

* Modern webpage layout structure
* Responsive design principles
* Flexbox-based alignment
* UI component structuring
* Styling using pure CSS (no frameworks)

The webpage includes navigation menus, hero sections, ride suggestions, planning features, and footer components similar to Uber’s official website.

---

## 🛠️ Technologies Used

* **HTML5** – Page structure and semantic layout
* **CSS3** – Styling and responsiveness
* **Font Awesome** – Icons integration
* **Flexbox** – Layout alignment

## 🚀 How to Run the Project

1. Download or clone the repository:

   ```bash
   git clone https://github.com/your-username/uber-landing-page.git
   ```

2. Open the project folder.

3. Double-click **index.html**
   OR open it in any browser:

   ```
   Chrome / Edge / Firefox
   ```

No installation or dependencies required.
